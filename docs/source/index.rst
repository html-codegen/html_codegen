HTML Codegen Documentation
==========================

HTML Codegen - это Python библиотека для программной генерации HTML документов с поддержкой Brython. 
Позволяет создавать HTML структуры через Python код с использованием контекстных менеджеров, 
динамического создания элементов и полной поддержки HTML5 семантики.

Основные возможности
--------------------

* **Контекстные менеджеры** - используйте ``with`` для создания иерархических HTML структур
* **Динамическое создание тегов** - генерируйте HTML теги через вызовы методов
* **HTML5 семантика** - полная поддержка всех семантических элементов HTML5
* **Интеграция с Brython** - выполнение Python кода в браузере
* **Типизация** - полная поддержка type hints и статической типизации
* **Модульная архитектура** - теги организованы по логическим категориям

Быстрый пример
--------------

.. code-block:: python

    from html_codegen import html, head, body, title, div, p
    
    with html() as doc:
        with head():
            title("Моя страница")
        with body():
            div().p().text("Привет, мир!")
